DESCRIPTION

In this lab, the user adjusts 4 input switches to control a 7 segment display component, 
3 LED lights based on logic from a truth table, and another set of 3 LED lights based on logic from 
a seperate truth table. 

-----------
FILES

-
Lab1.lgi

This file includes the MML logic simulation of the lab.

-----------
INSTRUCTIONS

This program is intended to be run using the schematic entry logic simulation program, Multimedia Logic.
Adjust the user input switches on page one of Lab1.lgi to run the simulations.
