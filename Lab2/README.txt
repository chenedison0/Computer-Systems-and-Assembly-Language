Lab 2: Simple Data Path

-----------
DESCRIPTION

In this lab, the user enters numbers, either through a keypad or an ALU output, into 4 registers and 
selects 2 to be inputs into an ALU. The ALU then shifts the second input by the number of times 
specified by the first input.

-----------
FILES

-
Lab2.lgi

This file includes the MML simulation of the lab.

-----------
INSTRUSTIONS

This program is intended to be run using the schematic entry logic simulation program, Multimedia Logic.
Adjust the user input switches on page one of Lab2.lgi to run the simulations.
