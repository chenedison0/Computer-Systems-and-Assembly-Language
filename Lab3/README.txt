Lab 3:  ASCII-risks (Asterisks)

-----------
DESCRIPTION

In this lab, the user enters a number greater than 0. The program will print variable-sized 
ASCII diamonds and a sequence of embedded numbers.
-----------
FILES

-
Lab3.asm

This file includes the assembly code of the lab.

-----------
INSTRUSTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator
(MARS). Follow the prompt in the RUN I/O message box after assembling and running the code.
